---
title: "数据库安装时报错：no such file or director"
---

**【问题描述】**

数据库安装时报错：`no such file or director`。

**【问题原因】**

该报错通常是由于安装包与操作系统版本或 CPU 架构不匹配、安装包不完整或损坏，或者操作系统为最小化安装缺少解压工具 `tar` 导致安装程序无法正常解压执行。

**【问题解决】**

- 检查安装包和操作系统版本以及 CPU 架构是否匹配；

- 对安装包进行完整性校验；

```
md5sum XXX.iso
```

- 查看操作系统是否为最小化安装，是否有解压工具 tar，如果没有则需要安装 tar 工具。

```
yum install tar -y
```
