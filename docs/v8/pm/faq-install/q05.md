---
title: "Linux 上无法启动管理工具"
---

**【问题描述】**

Linux 上无法启动管理工具。

**【问题原因】**

达梦数据库管理工具为图形化界面程序，依赖图形化界面环境及相应的显示权限，若系统未安装图形化界面或当前用户没有图形界面访问权限，会导致管理工具无法正常启动。

**【问题解决】**

- 保证安装时已安装图形化界面。

- 使用 `root` 用户执行命令 `xhost +` 登录 `dmdba` 用户执行命令 `export DISPLAY=:0.0` 再执行命令 `xhost +`

- 使用 `dmdba` 用户进入数据库安装目录下的 tool 目录执行如下命令打开管理工具
