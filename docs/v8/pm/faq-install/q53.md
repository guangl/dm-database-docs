---
title: "注册达梦数据库服务提示 -i 参数无效"
---

**【问题描述】**

注册达梦数据库服务提示 -i 参数无效。

**【问题原因】**

`-i` 参数是 DM7 的注册服务参数，如果用的是 DM8 数据库，则用 `-dm_ini` 参数，参数名不匹配导致提示参数无效。

**【问题解决】**

改用 `-dm_ini` 参数进行注册，例如：注册服务名为 `DmServiceDMSERVER` 的实例服务：
```
./dm_service_installer.sh -t dmserver -dm_ini /dm8/data/DAMENG/dm.ini -p DMSERVER
```
