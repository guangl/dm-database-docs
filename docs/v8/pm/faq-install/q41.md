---
title: "数据库安装前，是否需要预装 JDK？"
---

**【问题描述】**

数据库安装前，是否需要预装 `JDK`。

**【问题原因】**

该操作没有特定触发原因，属于安装/部署过程中的常规操作需求。用户在安装前通常需要确认是否要预先准备相关软件环境依赖，因此产生该疑问。

**【问题解决】**

不需要另外安装 `JDK`，达梦数据库自带 `JDK` 环境，软件环境安装要求如下：

| 软件环境 | 要求 |
| --- | --- |
| Windows | 简体中文服务器版 sp2 以上 |
| Linux | glibc2.3 以上，内核 2.6，已安装 KDE/GNOME 桌面环境，预装 UnixODBC 组件 |
| 网络协议 | TCP/IP |
| 系统盘 | 1 GB 以上剩余空间 |
