**【问题描述】**

**【问题分析】**

当前操作系统的登录用户应该无图形界面权限，如果在当前会话中启用图形界面需要将图形界面权限放开。

**【问题解决】**

- 确认当前操作系统环境安装有图形界面，切换到 `root` 用户后在命令行中输入 `xhost +`，可以使得 `dmdba` 可以调用图形界面进行安装。

- 可尝试使用 `./DMInstall.bin -i` 命令，通过字符界面进行安装。
