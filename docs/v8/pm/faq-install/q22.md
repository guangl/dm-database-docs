---
title: "初始化报错：create dir '/data/dmdata/DAMENG'failed"
---

**【问题描述】**

初始化报错：`create dir '/data/dmdata/DAMENG'failed`。

**【问题分析】**

出现该报错一般是由于初始化实例时指定的路径错误或不存在导致或者是由于该路径没有 `dmdba` 用户的读写权限导致。

**【问题解决】**

- 检查路径是否输入正确，等号前后不能有空格。

- 检查 `dmdba` 用户是否有对应路径的读写权限，如果没有可参考如下命令授予权限：
```
cd /data
chown -R dmdba:dinstall dmdata
chmod -R 755 dmdata
```

- 用 `ls -ll` 命令查看目录权限是否修改完成。

- 重新初始化实例。
