---
title: "文件不存在，请检查此 KEY 的路径是否正确"
---

**【问题描述】**

在安装数据库时报错：`文件不存在，请检查此 KEY 的路径是否正确`。

**【问题原因】**

该报错通常是由于授权文件（Key 文件）的命名格式或路径与安装程序的预期不符，例如未将文件重命名为 `dm.key`，或文件属主属组不正确，导致安装程序无法识别该 Key 文件。

**【问题解决】**

- 如果获取的授权文件格式为 `dmxxxxxxx.key`，则在安装的时候需要把 Key 文件名称修改为 `dm.key` 并且通过以下命令修改属主和属组。
```
chown dmdba:dinstall dm.key
```

- 如果是从达梦在线服务平台下载试用数据库，可以在输入 Key 文件路径时，选择 `NO`，继续进行安装。
