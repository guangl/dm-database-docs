---
title: "未找到 tar 命令"
---

**【问题描述】**

未找到 `tar` 命令。

**【问题原因】**

`tar` 是 Linux/Unix 系统下的打包压缩工具，用于将多个文件合并为一个文件，缺失该命令会导致数据库无法安装。系统报错找不到 `tar` 命令，可能是由于以下原因造成：

1. 未安装 `tar` 包：部分系统在最小化安装模式下默认不安装 `tar` 命令，可通过部署 `YUM` 源安装该包。
2. `tar` 命令路径不在系统环境变量中：如果已安装但仍提示找不到命令，可能是该命令的路径未加入系统环境变量。可编辑配置文件添加路径，例如在 `.bashrc` 文件中加入 `export PATH=$PATH:/usr/bin/tar`，然后执行 `source .bashrc` 使其生效。

**【问题解决】**

- 检查 `tar` 命令是否已安装，如未安装，请先安装；
- 如已安装但仍无法使用，请检查系统环境变量配置是否正确。
