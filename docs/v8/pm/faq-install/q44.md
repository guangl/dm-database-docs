---
title: "数据库安装报错:Error occurred during initiazation of VM&Unable to load native library"
---

**【问题描述】**

数据库安装的时候报错：

- `Error occurred during initiazation of VM`

- `Unable to load native library`

**【问题原因】**

该报错通常是由于操作系统缺少安装程序运行所依赖的某些系统库文件（如 `libnsl`），导致虚拟机初始化或本地库加载失败。

**【问题解决】**

可通过尝试创建软连接来解决该问题。
```
ln -s /usr/lib64/libnsl.so.2  /usr/lib64/libnsl.so.1
```
