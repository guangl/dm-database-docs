# 许可证（License）的安装

> [!note]
> 1. 替换原有文件前，建议先备份。
> 2. 替换原有文件前，先要判断 License 架构是否满足现有架构。
> 3. 替换原有文件前，需要匹配操作系统及 CPU 架构。

操作步骤：

1. **定位服务器目录**
   - 以 `dmdba` 登录系统。
   - 启动终端，进入安装目录，例如：`cd /opt/dmdbms/bin`。

2. **复制许可证文件**
   - 将 `dm.key` 拷贝到该目录。

3. **验证和查看许可证**
   - 执行 `SP_LOAD_LIC_INFO();` 进行校验。
   - 执行 `SELECT * FROM V$LICENSE;` 查看许可证信息。

## 关键特性

- `dm.key` 的放置不受服务器运行状态影响。
- 替换许可证文件不会影响正在运行的 `DMServer`。
- 服务器会定时检测并读取 `dm.key` 中的信息。
