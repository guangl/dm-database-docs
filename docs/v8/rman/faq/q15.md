---
title: "Windows 下配置归档目录报错：归档目标无效"
---

**【问题描述】**

举例如下：

```sql
ALTER DATABASE ADD ARCHIVELOG 'DEST = d:/test, TYPE = local, FILE_SIZE = 1024, SPACE_LIMIT = 2048';
```

报错：归档目标无效。

**【问题原因】**

`DEST` 路径中驱动器盘符大小写不正确（写成了小写 `d:`），导致归档目标路径解析无效。

**【问题解决】**

```sql
ALTER DATABASE ADD ARCHIVELOG 'DEST = D:/test, TYPE = local, FILE_SIZE = 1024, SPACE_LIMIT = 2048';
```

修改 DEST 路径，建议选择路径的时候直接复制文件路径，不要自己写，避免写错。
