---
title: "备份报错 [-8084]: 起始 LSN [47619] 小于等于基备份结束 LSN [47631]"
---

**【问题描述】**

执行增量备份时报错：`[-8084]: 起始 LSN [47619] 小于等于基备份结束 LSN [47631]`。

**【问题原因】**

未找到基备份，如果未指定基备份，达梦数据库会去默认的路径下查找基备份。

**【问题解决】**

如果未指定基备份，达梦数据库会去默认的路径下查找基备份，默认路径可执行以下语句：

```sql
select para_name,para_value from v$dm_ini where para_name='BAK_PATH';
```

增量备份时指基备份集，使用以下命令：

```sql
backup database increment with backupdir '基备份路径' backupset '备份路径';
```

同时注意全量备份后需要对数据操作一下，例如加一条数据之类的，要不然没有增量，自然不能增量备份。
