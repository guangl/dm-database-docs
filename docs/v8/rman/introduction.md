# dmrman

dmrman 是达梦数据库的命令行备份还原工具，主要用于执行脱机备份与脱机还原操作。脱机备份/还原是指数据库处于关闭状态下进行的备份还原操作；脱机还原操作的目标库也必须处于关闭状态。只有已经关闭的数据库才允许执行脱机备份，正在运行的数据库无法通过 dmrman 执行脱机备份。

从 V2.0 版本开始，脱机备份均通过 dmrman 工具完成，并支持对异常关闭的数据库进行脱机库备份。备份异常关闭的数据库要求已配置本地归档，且本地归档完整；如果本地归档不完整，则需要先修复本地归档，再进行备份。

dmrman 可以对库备份、表空间备份和归档备份执行脱机还原；物理还原也一般通过 dmrman 工具（或 SQL 语句）完成，将备份集中的数据内容重新拷贝、写入目标文件。脱机场景下备份还原的实现策略（dmap 辅助进程方式或无辅助进程方式）通过 dmrman 工具的 `use_ap` 参数指定，对应联机操作中 dm.ini 的 `bak_use_ap` 参数。

该工具位于安装目录的 `/bin` 目录下。
