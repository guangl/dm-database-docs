# AUTO 事务

控制自动提交事务的行为。

## 语法

## 参数

- `COMMIT` - 可选，用于显式说明事务模式，默认为隐式
- `ON` - 启用自动提交，每条语句执行完毕后自动提交
- `OFF` - 禁用自动提交，需要手动执行 COMMIT 提交事务

## 示例

### 启用自动提交

```sql
SET AUTO ON;
```

### 禁用自动提交

```sql
SET AUTO COMMIT OFF;
```

## 说明

- 当 AUTO 为 ON 时，每条 SQL 语句执行完成后自动进行事务提交
- 当 AUTO 为 OFF 时，需要显式执行 COMMIT 语句来提交事务
- 默认值为 ON
