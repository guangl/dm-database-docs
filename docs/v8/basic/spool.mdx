# SPOOL 输出

设置 SQL 输出的目标位置，支持输出到文件或关闭输出。

## 语法

## 参数

- `SPO[OL]` - 可使用缩写 `SPO`，完整形式为 `SPOOL`
- `<file>` - 输出文件路径
- `OFF` - 关闭文件输出，取消将输出重定向到文件
- `CRE[ATE] | REP[LACE] | APP[END]` - 可选，文件操作模式
  - `CREATE`（缩写 `CRE`）- 创建新文件，文件存在时报错
  - `REPLACE`（缩写 `REP`）- 替换已存在的文件
  - `APPEND`（缩写 `APP`）- 追加到文件末尾
- `NO_PRINT` - 可选，执行命令但不打印结果

## 示例

### 创建新输出文件

```sql
SPOOL /tmp/output.sql CREATE;
```

### 替换文件

```sql
SPOOL output.txt REPLACE;
```

### 追加到文件

```sql
SPO result.log APP NO_PRINT;
```

### 关闭输出

```sql
SPOOL OFF;
```

## 说明

- 使用缩写时，方括号 `[]` 内的部分为可选
- 每个模式选项都有对应的缩写形式
- `NO_PRINT` 常用于脚本中，避免屏幕输出过多内容
