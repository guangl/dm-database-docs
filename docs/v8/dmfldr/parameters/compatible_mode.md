---
title: COMPATIBLE_MODE
---

兼容模式，可取值 0、1、2，缺省为 0。0：不做兼容；1：兼容 DB2；2：兼容 MYSQL。

兼容 DB2 时支持功能：

1. 字符串列封闭符默认为 `"`，全局封闭符只针对字符串列生效；
2. 大字段导入导出文件的数据文件格式更换为：`filename.nnn.mmm/`，其中 filename 是大字段文件名，nnn 是文件偏移（取值范围 0~BIGINT），mmm 是大字段长度（取值范围 0~BIGINT）；
3. 支持 `PRIORITY_ENCLOSE` 参数；
4. 数据没有封闭，则被认为是空值，存在封闭符但没有数据，则认为是空串。

兼容 MYSQL 时，支持 MYSQL 中对 NULL 值、空串以及转义符的处理。此参数为可选参数。
