# dmfldr

达梦数据库提供两种快速装载工具形式，用于在达梦数据库中进行表数据的快速导入和导出操作：

- **dmfldr**：适用于软硬件资源充足的场景，在单台机器上独立运行，完成装载任务。
- **dmldrc 和 dmldrp**：轻量级方案，分别部署在两台不同机器上，适合资源匮乏的极端情况。

支持的对象范围：

- 表及表的同义词：支持数据导入和导出。
- 视图及视图的同义词：仅支持数据导出。

## 系统结构

### dmfldr 结构

dmfldr 由客户端和服务器端模块组成：

- **客户端**：负责初始化装载环境、读取数据、打包数据、发送数据。
- **服务器模块**：嵌入数据库服务器，执行实际的装载操作。

导入过程：客户端接收命令、分析文件，将数据打包后发送给服务器完成装载。

导出过程：客户端向服务器发送请求，服务器打包数据并返回，客户端将数据写入文件。

### dmldrc 和 dmldrp 结构

轻量级工具套装分为：

- **dmldrc**：客户端，负责初始化和数据处理。
- **dmldrp**：服务器端，负责任务处理和数据发送。

两个工具可以部署在不同机器上，分散系统负载，减轻硬件压力。
